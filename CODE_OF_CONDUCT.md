# Code of Conduct

## Our Commitment

We are committed to a respectful, inclusive, and harassment-free community for everyone contributing to Aegis C2.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Focus feedback on code, design, and outcomes.
- Help keep discussions technical, actionable, and professional.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language.
- Personal attacks, intimidation, or repeated hostile behavior.
- Publishing private or sensitive information without explicit permission.
- Disruptive trolling, bad-faith participation, or spam.

## Scope

This Code of Conduct applies to project spaces including:

- issues and pull requests
- code review discussions
- project communication channels

## Enforcement

Maintainers may remove or reject comments, commits, issues, and contributions that violate this Code of Conduct.

Depending on severity, maintainers may also restrict participation.

## Reporting

If you experience or witness unacceptable behavior:

- report to maintainers privately
- include relevant context, links, and timestamps where possible

For security-sensitive reports, follow `SECURITY.md`.
